# Autofill.pw - secure and random password generator

Autofill.pw is a client-side, secure random password generator. It's designed with privacy in mind, performing all operations directly in your browser. No data, especially your generated passwords, is ever collected, stored, or transmitted.

## Features

* **Client-Side Generation:** All passwords are generated directly in your browser. Nothing is sent to any server.
* **Customizable Length:** Generate passwords from 8 to 64 characters long.
* **Character Set Control:**
    * Include Uppercase letters (A-Z)
    * Include Lowercase letters (a-z)
    * Include Numbers (0-9)
    * Include Symbols (!@#$%)
* **Cyrillic Support:** Optionally mix Latin characters with Serbian Cyrillic characters for added complexity.
* **One-Click Copy:** Easily copy the generated password to your clipboard.
* **Dark/Light Mode:** Choose your preferred theme, which is saved in your browser's local storage.
* **Multi-Language Support:**
    * Currently available in English (EN) and Serbian latin (SR).
    * Language preference is saved in local storage.
    * Detects browser language for initial setup (in theory).
* **Responsive Design:** Works smoothly on desktop and mobile devices.

## Privacy First

Your privacy is very important. Autofill.pw operates entirely within your web browser.
* **No Data Collection:** Website does not collect, store, or transmit any personal data, including, and especially the passwords you generate.
* **No Server Interaction for Generation:** Password generation is done locally on your device. You can even disconnect, once the page loads.
* **Local Storage for Preferences:** Theme and language preferences are stored in your browser's local storage only. Never sent to the server.

## How to Use

1.  **Visit the website:** [autofill.pw](https://autofill.pw/), or run a local instance!
2.  **Adjust Options:**
    * Use the slider to set your desired **Password length**.
    * Toggle the switches for **Character types** (Uppercase, Lowercase, Numbers, Symbols).
    * Optionally, enable "Mix with Cyrillic" for Serbian Cyrillic characters.
3.  **Generate:** Click the "Generate password" button.
4.  **Copy:** Click the "Copy" button to copy the password to your clipboard.

You can also switch the theme (light/dark) using the sun/moon icon and change the language using the "EN" / "SR" buttons at the top.
